Forest Fire Monitoring System

This project is a simple simulated forest monitoring system.

Files:
1. sensor_simulator.py
   Simulates environmental data collection.
   Uses a digital start button and analogue environmental values.

2. wildfire_model.py
   Calculates wildfire risk from collected data.
   Produces an adaptive alert output.

3. scenarios.py
   Runs two what-if simulations:
   - hotter weather
   - drier soil

4. dataset.csv
   Environmental data collected by the system.

5. processed_data.csv
   Output after wildfire risk calculation.

6. scenario_results.csv
   Output from the two what-if simulations.

How to run:
1. Run sensor_simulator.py
2. Run wildfire_model.py
3. Run scenarios.py
